# NestJs에서 DB 사용하기

# 전체 명령어
-> yarn add @nestjs/typeorm typeorm pg

1. @nestjs/typeorm
-> nestjs에서 typeorm을 이용하기 위한 유틸리티

2. typeorm
-> typeorm 패키지

3. pg
-> pgadmin 패키지

# TypeOrmModule.forRoot, TypeOrmModule.forFeature
-> forRoot는 app.module(main)에 db 설정 내용을 작성
-> forFeature는 모델에 해당하는 repository를 작성
